# Migrando si estás en un pozo

Si está en un pozo antes de la migración de _front end_ (En la cual solo habían 9 personas) aca están los pasos para resolver.

Vaya a la dirección del contrato antiguo:

https://etherscan.io/address/0x02444a8fEeaD1a2E78da0e2a157f4C7B848aB021#writeContract

Vaya al final y de clic en "Withdraw"

<=== IMAGE HERE: Withdraw screenshot ===>

Ingrese el ID del pozo para PID.

Ingrese el monto convertido a wei. Por ejemplo, para remover un token LP deberá ingresar 1000000000000000000

Puede usar  https://eth-converter.com/ para calcular el número de tokens a retirar.

Para un referido necesitará fijar 0x0000000000000000000000000000000000000000

Si una tarifa se aplica vaya a nuestro Discord y envíe un mensaje a BaoMan con la TX para poder reembolsar la tarifa.
